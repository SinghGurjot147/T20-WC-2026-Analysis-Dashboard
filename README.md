# T20-WC-2026-Analysis-Dashboard


This project focuses on analyzing the ICC Men’s T20 World Cup 2026 dataset and transforming raw cricket data into meaningful insights using Microsoft Excel. The primary objective is to explore player performance, team statistics, and match outcomes through effective data analysis and visualization techniques.

The dataset, collected from reliable sources such as Kaggle, ESPNcricinfo, and the ICC official website, includes detailed information about matches, batting and bowling statistics, and tournament results . It covers 20 teams, 55 matches, and multiple venues, providing a comprehensive view of the tournament.

The project follows a structured data analysis workflow, starting with data cleaning and preprocessing. Inconsistent values were corrected, missing data was handled, and formatting issues were resolved using Excel functions. The dataset was then transformed by creating additional columns such as match type (defending/chasing) and toss impact to enhance analysis.

Pivot tables were used to summarize large datasets and identify key metrics like top batsmen, leading bowlers, and match distributions. Various charts, including bar charts and donut charts, were created to visually represent trends and comparisons.

The final output is an interactive dashboard that presents insights in a clear and user-friendly format . Features such as slicers allow dynamic filtering, enabling users to explore data efficiently.

Key insights from the analysis include patterns in match-winning strategies, player consistency, and the limited impact of toss on match results. Overall, this project demonstrates how Excel can be effectively used as a powerful tool for data analysis and visualization in real-world scenarios.
